> 개인 프로젝트

- 프로젝트 명 : "반려" ( PetShop ) <br>
- 인 원 : 1 명 <br>
- 기 간 : 2022.06.27 ~ 2022.07.08 <br>
- DBMS : Oracle 11g Express Edition <br>
- 언어 : Java SE 8 Platform, HTML5, CSS, CSS3, JavaScript <br>
- 패턴 : MVC 패턴 <br>
- 오픈 Library : Java Script jquery-3.4.1, jquery-ui-1.12.1, cos-2022.05  <br>
- 프로젝트 소개 : 프로젝트 소개 <br>
